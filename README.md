# console-photo-album
<!-- ![console-photo-album](./preview.png) -->

A Node.js console app to display photo albums from JSON Placeholder API https://jsonplaceholder.typicode.com/

# Set Up
> Run the following commands from the root of this repository
```
npm i
```
```
npm run dev
```

# Testing
> Run the following commands from the root of this repository
```
npm run test
```
